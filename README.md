# NBC Maritime Ltd. - Static Company Profile

Clean, slide-based presentation website for NBC Maritime Ltd., Nigeria's premier ship management company.

## Overview

Static website with 8 slides presenting NBC Maritime's corporate profile, services, team, and global network. Based on client's PowerPoint deck with web-optimized design.

## Features

- **Slide Navigation**: Full-screen slides with side navigation (01-08)
- **White Backgrounds**: Clean corporate aesthetic per client request
- **Brand Colors**: #2875a8 (NBC Blue) throughout
- **Country Flags**: Emoji flags for global ABC Group locations
- **Responsive**: Mobile-optimized layout
- **Static**: Pure HTML/CSS, no dependencies

## Structure

### Slides:

1. **Hero** - Company tagline and services
2. **Global Locations** - ABC Group network with flag emojis
3. **Company Profile** - Vision & Mission
4. **About Us** - Services and compliance details
5. **Value Proposition** - What We Offer / Why Choose Us
6. **Key People** - Team contacts (no photos per client request)
7. **Vetted By** - Certifications and trust badges
8. **Thank You** - Contact information

- **Body**: Open Sans (400-600 weight)
- **Eyebrows**: 0.85rem, 700 weight, uppercase
