# NBC Maritime Ltd.

Static corporate website for NBC Maritime - Premier ship management company in West Africa.

## Overview

This is a clean, corporate website based on ABC Maritime AG's design approach, adapted for NBC Maritime with Nigerian focus.

## Features

- **Clean Design**: White backgrounds, professional typography
- **Responsive**: Mobile-first approach
- **Static**: No framework dependencies, pure HTML/CSS
- **Fast**: Minimal assets, optimized loading
- **Accessible**: Semantic HTML, proper heading structure

## Structure

- `index.html` - Main page with all sections
- `styles.css` - All styling in one file
- Uses system fonts + Google Fonts (Inter + Poppins)

## Color Scheme

- Primary: #2875a8 (NBC Blue)
- Text: #111827 (Gray 900)
- Backgrounds: White (#ffffff) and Light Gray (#f9fafb)

## Sections

1. Hero - Company tagline and services
2. Stats - Key metrics
3. Global Locations - ABC Group network with flags
4. Company Profile - Vision & Mission
5. About & Services - Detailed information
6. Value Propositions - What we offer / Why choose us
7. Key People - Team with placeholder SVGs
8. Trusted By - Certifications and endorsements
9. Contact - Office locations
10. Footer - Navigation and info
