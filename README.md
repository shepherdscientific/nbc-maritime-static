# NBC Maritime Ltd. - Company Profile

Clean, slide-based static website for NBC Maritime ship management company.

## Overview

This is a presentation-style website with 8 full-screen slides, matching the company's PowerPoint deck aesthetic. Built with pure HTML/CSS - no frameworks, no dependencies.

## Design Principles

✅ **WHITE backgrounds** (per client request - no cream/beige)
✅ **#2875a8 brand color** throughout
✅ **Slide-based navigation** with numbered dots (01-08)
✅ **Country flag emojis** for global ABC Group locations
✅ **NBC Nigeria prominently highlighted** in locations grid
✅ **NO team photos** (per client request - text only)
✅ **Professional, corporate aesthetic**
✅ **ABC Maritime design reference**

## Structure

### Slides

1. **Hero** - Company tagline and services
2. **Global Locations** - ABC Group network with flags (Nigeria highlighted)
3. **Company Profile** - Vision & Mission cards
4. **About Us** - Services and compliance details
5. **Value Propositions** - What we offer / Why choose us
6. **Key People** - Team contacts (text only, no photos)
7. **Vetted By** - Trust indicators
8. **Thank You** - Contact information

### Features

- **Scroll-snap navigation** - Each slide is full-screen
- **Side navigation dots** - Quick jump to any slide (01-08)
- **Mobile responsive** - Adapts to small screens
- **Smooth animations** - Subtle hover effects
- **Fast loading** - No external dependencies except Google Fonts

## Color Scheme

```css
--primary: #2875a8        /* NBC Blue */
--primary-dark: #1a5580   /* Darker blue for gradients */
--primary-light: #4a94c4  /* Lighter blue for highlights */
--white: #ffffff          /* Pure white backgrounds */
--line: #d9e0ea          /* Border color */
```

## Typography

- **Headings:** Poppins (700, 800)
- **Body:** Open Sans (400, 600, 700)
- **Responsive sizing:** clamp() for fluid typography
